This directory contains the MIT-P scalar field dataset. It is a regional subset of a global P-wave seismic tomography model from Li et al. (2008). 

Please use the following citation for this dataset:

Li, C., van der Hilst, R., Engdahl, E., and Burdick, S., 2008, A new global model for P wave speed variations in Earth's mantle: Geochemistry, Geophysics, Geosystems, v. 9, no. 5, p. 21. (DOI: 10.1029/2007GC001806)

The directory contains two files:
- MITP08-regional.gproj - GPlates project file (most convenient way to load the 3D visualisation)
- MITP08-regional.gpml - native GPlates Markup Language .gpml format
- MITP08-regional.gpsf - GPlates scalar field file


To load these datasets in GPlates do the following:

1.  Open GPlates
2.  Pull down the GPlates File menu and select the operation Open Feature Collection
3.  Click and select the GPROJ or GPML file (GRPOJ recommended for novice user)
4.  Click Open 

Alternatively, drag and drop the GRPOJ or GPML file onto the globe.

For more information, read the GPlates manual which can be downloaded from www.gplates.org.
