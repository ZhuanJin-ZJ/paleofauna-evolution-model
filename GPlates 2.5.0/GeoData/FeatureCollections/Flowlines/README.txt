This directory contains examples of flowlines in the Atlantic Ocean.  The directory has a .gpml file which contains seed points at several locations along the mid-Atlantic Ridge.  When loaded with a rotation file, flowlines will be drawn reflecting the relative motion between the two plate pairs along the mid ocean ridge.

The directory has 1 file:
- AtlanticFlowlines.gpml

To load these datasets in GPlates do the following:

1.  Open GPlates
2.  Pull down the GPlates File menu and select the operation Open Feature Collection
3.  Click the file you want to load
4.  Click Open 

Alternatively, drag and drop the files onto the globe.

The file contains seed points at several locations along the mid-Atlantic Ridge.  When loaded with a rotation file, flowlines will be drawn reflecting the relative motion between the two plate pairs along the mid ocean ridge.

Play around with the GPlates buttons to make an animation, select features, draw features, etc.  For more information, read the GPlates manual which can be downloaded from www.gplates.org.
