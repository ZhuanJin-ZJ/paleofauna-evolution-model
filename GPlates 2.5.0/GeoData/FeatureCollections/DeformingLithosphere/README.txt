This directory contains the Zahirovic et al. (2022) topological plate model - with namely the Müller et al. (2019) and Cao et al. (2020) deforming regions.

Directory contains several files:

- Zahirovic_etal_2022_DeformingLithosphere.gproj - GPlates project file
- Zahirovic_etal_2022_Deforming_Networks_Active.gpml - Active deforming zones
- Zahirovic_etal_2022_Deforming_Networks_Inactive.gpml - Inactive deforming zones, needed to bring points to the present day 
- Zahirovic_etal_2022_Deforming_Points_L7.gpmlz - deforming points and scalar fields 
- stretching_factor_muller_etal_2019.cpt - help with applying the stretching or contraction factor colour legend from Müller et al. (2019).


To load these datasets in GPlates do the following:

1.  Open GPlates
2.  Pull down the GPlates File menu and select the operation Open Project
3.  Click the LowRes project file
4.  Click Open (it will take a few minutes to load)

Alternatively, drag and drop the project onto the globe. For more information on using the deforming model, see the GPlates tutorials.
