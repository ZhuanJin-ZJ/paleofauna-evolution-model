This directory contains the GPlates convenience dataset of perceptually uniform scientific colour maps from Crameri et al. (2020). This is version 7 downloaded from https://www.fabiocrameri.ch/colourmaps/

Reference:
Crameri, F., G.E. Shephard, and P.J. Heron, 2020, The misuse of colour in science communication, Nature Communications, 11, 5444, doi:10.1038/s41467-020-19160-7.
